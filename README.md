# Better Spectate Player Highlighting
A small mod that lets you highlight players separately per team.

Improves upon the vanilla "Highlight Players" keybind. The vanilla key highlights all players and works exclusively in spectator mode. With this mod you can instead separately apply glowing to each outline color, swap between toggle/hold mode, and see glowing/invisibility on highlighted players through a brighter or darker color. This mod's highlighting also works at any time the player has creative flight, instead of only when they are in spectator mode.
